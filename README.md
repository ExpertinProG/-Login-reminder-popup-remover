# Login Reminder Popup Remover

## Description
This user script removes nagging login popups and banners from various social media platforms, including Facebook, Instagram, Reddit, Twitter, Quora, Ask.fm, VK, Pinterest, Tumblr, Twitch, and the mobile versions of YouTube and TikTok.

## Installation
To install the user script, follow these steps:
1. Install a user script manager like Tampermonkey or Greasemonkey.
2. Add the script to your user script manager.

## Usage
1. Navigate to a supported platform.
2. The script will automatically remove login popups and banners.

## Features
- **Multi-language Support**: Available in English and Italian.
- **Automatic Domain Update**: Checks and updates the script for new domains.
- **Popup Removal**: Removes login popups and banners from supported platforms.
- **Customizable**: Users can enable or disable the script for specific sites.

## Supported Platforms
- **Social Media**: Facebook, Instagram, Reddit, Twitter, Quora, Ask.fm, VK, Pinterest, Tumblr, Twitch.
- **Video Platforms**: YouTube (mobile), TikTok (mobile).
